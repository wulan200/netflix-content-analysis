# Netflix Content Analysis

## Project Overview
This project analyzes Netflix's content catalog using Python to identify trends in content types, genres, countries, ratings, and release years.

## Objectives
- Understand the distribution of Movies and TV Shows.
- Identify the most popular genres on Netflix.
- Analyze content distribution across countries.
- Explore release year trends.
- Examine content ratings.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab

## Data Cleaning
- Filled missing values in director, cast, country, and rating columns.
- Removed records with missing date_added and duration values.

## Key Findings
- Movies make up the majority of Netflix content.
- The United States contributes the largest amount of content.
- Netflix content production increased significantly after 2015.
- TV-MA is among the most common content ratings.
- Drama and International Movies are highly represented genres.

## Visualizations
The project includes:
- Movie vs TV Show Distribution
- Top Genres Analysis
- Top Countries Analysis
- Content Release Trend
- Rating Distribution
